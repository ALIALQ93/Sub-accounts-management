# الفواتير

## الهدف الواجب الوصول إليه

أن تكون **مستند المصدر** للبيع والشراء: تُنشئ ذمماً أو نقداً، تُحدّث المخزون (لاحقاً)، وتُغذّي **إغلاق الحركات** والتقارير.

**فلسفة الأنماط:** ضبط **الإعدادات الافتراضية** — أنماط جاهزة للتسريع (قابلة للتعديل، غير إلزامية). المحور الجوهري لكل فاتورة: **إدخال** أو **إخراج**؛ الإعدادات تحدد التعامل مع **المواد والحسابات**.

## النتيجة المرجوة

- أنماط فواتير غير محدودة (مثل السندات)
- كل نمط: اتجاه (إدخال/إخراج) + حسابات + سلوك مواد
- كل فاتورة: **تبويب مواد** + **تبويب حسابات إضافية** (مصاريف نقل، خدمات…)
- تحصيل عبر **قبض/دفع عادي** أو **إغلاق حركات** (مساران منفصلان)
