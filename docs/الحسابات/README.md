# الحسابات

## الهدف الواجب الوصول إليه

أن تكون **الطبقة المحاسبية المركزية**: دليل حسابات، عملات، مراكز كلفة، دفاتر، موازنات، وقوائم مالية — مصدر الحقيقة لكل التقارير.

## النتيجة المرجوة

- حركة كل حساب موثّقة ومتزنة
- تعدد عملات مع أساس موحّد
- تتبع تكلفة بمراكز الكلفة
- إقفال فترات وقوائم ختامية
