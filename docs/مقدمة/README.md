# مقدمة

## الهدف الواجب الوصول إليه

توفير نقطة دخول واضحة للمستخدم: ترحيب، فهم إصدار النظام، دليل تشغيل، ومساعدة — دون الخلط بين «التجربة» و«الإنتاج».

## النتيجة المرجوة

- المستخدم يعرف **أي إصدار** يعمل عليه وما الجديد
- مسار سريع لبدء العمل (اختصارات، checklist إعداد)
- توثيق مساعدة مدمج أو مرتبط

## العلاقات

### يعتمد على

- إعدادات الشركة (`ملف`)
- إصدار التطبيق (`web/src/config/app-release.ts`)

### يغذّي

- كل الأقسام (توجيه المستخدم)

### يرتبط بـ

- [أدوات](أدوات/README.md) — صيانة واستيراد
- [ملف](ملف/README.md) — إعداد أولي
