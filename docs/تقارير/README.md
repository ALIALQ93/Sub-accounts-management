# تقارير

## الهدف الواجب الوصول إليه

تجميع بيانات كل الأقسام في **تقارير محاسبية وتشغيلية** قابلة للطباعة والتصدير واتخاذ القرار.

## النتيجة المرجوة

- مركز تقارير (`/reports`) بكل التقارير المصنّفة
- مقارنة فترات وتصدير PDF/Excel
- تقارير لكل قسم (مواد، سندات، CC…)

## العلاقات

### يعتمد على

- كل الأقسام التي تُنتج بيانات

### يغذّي

- الإدارة، [مقدمة](مقدمة/README.md) (لوحة تحكم)
