# الأصول الثابتة

## الهدف الواجب الوصول إليه

تسجيل **الأصول طويلة الأجل**، حساب **الإهلاك**، وتتبع العمليات (نقل، صيانة) مع ترحيل قيود تلقائية.

## النتيجة المرجوة

- بطاقة لكل أصل مع فئة وموقع
- إهلاك دوري → قيود يومية
- تقارير جرد وقيمة دفترية
