# المواد والمستودعات

## الهدف الواجب الوصول إليه

إدارة **الأصناف والمخزون** من التعريف حتى الحركة والتقارير، مع ربط التكلفة بالمحاسبة والفواتير.

## النتيجة المرجوة

- كل مادة لها بطاقة (وحدات، أسعار، باركود، مجموعات)
- مخزون دقيق لكل مستودع/فرع
- تقارير حركة، نواقص، أرباح، راكد
- ربط تلقائي بفواتير البيع/الشراء وقيود التكلفة
