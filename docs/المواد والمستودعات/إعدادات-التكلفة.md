# إعدادات التكلفة — المواد العامة

> **الموضوع:** **كيف** تُحسب تكلفة الوحدة (متوسط، FIFO…) — ضمن [إعدادات المخزون العامة](إعدادات-المخزون-العامة.md).  
> **قرار:** يُختار مع الإعداد الأولي — **غير قابل للتعديل** بعد العمليات.  
> **آخر تحديث:** 2026-07-05

---

## 1) الهدف

تحديد **خوارزمية التكلفة** — متى تُحدَّث تكلفة المادة عند الإدخال والإخراج والمناقلة.

| البند | القرار |
|--------|--------|
| مكان الإعداد | [الإعدادات العامة عند التثبيت](../ملف/الإعدادات-العامة-عند-التثبيت.md) — الخطوة ③ |
| القفل | نفس قاعدة §1 في إعدادات المخزون — لا تغيير بعد الحركات |
| التأثير | كل فواتير [الإدخال/الإخراج/المناقلة](../الفواتير/تعريف-الفواتير.md) |

---

## 2) أنظمة التكلفة

| النظام | الوصف | ملاحظة |
|--------|--------|--------|
| **متوسط مرجح** | متوسط يُحدَّث مع كل إدخال | شائع — يتفاعل مع إعادة التكليف عند النقل بأجور |
| **FIFO** | طبقات — أول صادر أول وارد | يتطلب تتبع دفعات |
| **معياري** | تكلفة ثابتة + فروقات | تصنيع |
| **آخر سعر شراء** | بسيط | أقل دقة |

---

## 3) تفاعل الفواتير مع التكلفة

| نوع الفاتورة | الاتجاه | تأثير التكلفة |
|--------------|---------|----------------|
| مشتريات | إدخال | تحديث متوسط/طبقة |
| مبيعات | إخراج | صرف تكلفة (قيد فوري إن **جرد مستمر**) |
| مرتجع مبيعات | إدخال | استرجاع تكلفة الصرف |
| **مرتجع مشتريات** | **إخراج** | عكس الشراء — تعديل رصيد التكلفة |
| **مناقلة** | transfer | نقل تكلفة؛ **+ تكلفة إضافية** (دوري ومستمر) |
| بضاعة أول المدة | إدخال | تكلفة افتتاحية يُدخلها المستخدم |

---

## 4) فصل التكلفة (مستودع / CC)

> التفاصيل الكاملة: [إعدادات-المخزون-العامة.md](إعدادات-المخزون-العامة.md) §6.

| الإعداد | النتيجة |
|---------|---------|
| `cost_per_warehouse = true` | تكلفة الوحدة **قد تختلف** بين المستودعات |
| `cost_per_warehouse = false` | تكلفة **موحّدة** — أجور النقل تعيد التكليف في **كل** المستودعات |
| `cost_per_cost_center = true` | تكلفة تُتابَع per مركز كلف |

---

## 5) الربط المحاسبي (جرد مستمر)

| الحركة | قيد نموذجي |
|--------|------------|
| شراء | مدين مخزون — دائن مورد |
| بيع | مدين تكلفة مبيعات — دائن مخزون (+ قيد إيراد) |
| مناقلة | مدين مخزون هدف — دائن مخزون مصدر |
| مصاريف نقل (دوري) | مدين مخزون/مصروف نقل — دائن نقد/ذمم |

في **الجرد الدوري** — قيود المبيعات والمناقلة تختلف؛ انظر [§2–§4](إعدادات-المخزون-العامة.md).
