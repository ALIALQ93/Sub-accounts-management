# سندات

## الهدف الواجب الوصول إليه

أن يكون **المدخل الرئيسي للمعاملات المالية**: كل قبض ودفع وتسوية وإغلاق ذمم يمر بمستند سند بحالة واضحة وقيد عند الترحيل.

## النتيجة المرجوة

- مسار: مسودة → اعتماد → ترحيل (أو تلقائي)
- أنواع متعددة حسب النشاط
- ربط بالعملاء/الموردين والتخصيصات
- مراقبة وتعديل وعكس عند الحاجة
