## إغلاق الحركات — قرار: تطوير كبير

| الميزة | الحالة | وثيقة |
|--------|--------|--------|
| `open_amount` حقيقي | 🟡 واجهة | `open_items_view` + `open-movements-api` |
| مدين + دائن مفتوح | 🟡 واجهة | فلتر جانب في إغلاق الحركات |
| مقاصة مراكز كلف | 🟡 واجهة | `voucher_netting_lines` + لوحة مقاصة |
| فروع + مقاصة فرع | 🟡 واجهة | فلتر فرع + مقاصة فرع |
| مقاصة تشمل نقد (افتراض CC) | 🟡 واجهة | `netting_includes_cash_default` per CC |
| استحقاق وتغطية جزئية | 🟡 واجهة | عرض استحقاق + فلتر مؤهل/متأخر |
| حسابات وسيطة | افتراض نمط/CC؛ تعديل في السند | ⚪ معتمد | §4.6 |
| لا يستبدل قبض/دفع | مساران منفصلان | ✅ قرار | قرارات #8 |
| يعتمد فواتير | [تعريف الفواتير](../الفواتير/تعريف-الفواتير.md) **معتمد** | 🟡 | بناء الفواتير |
