# ملف — المزايا

| الميزة | الوصف | العلاقات | الحالة | معيار الاكتمال | أين في الكود |
|--------|--------|----------|--------|----------------|--------------|
| الشركات | إدارة أكثر من شركة في قاعدة واحدة | كل الأقسام | ⚪ | `companies` + عزل بيانات | `company_settings` (شركة واحدة) |
| الفروع | تعريف فروع وربطها بالحركات | حسابات، مواد، POS | ⚪ | `branches` + `branch_id` على المعاملات | — |
| المراسلة | سجل مراسلات أو مراسلات داخلية | — | ⚪ | يُحدَّد عند الشرح | — |
| تصدير بيانات نقطة المبيع | تصدير مبيعات POS | نقاط المبيع | ⚪ | يعتمد بناء POS | — |
| استيراد بيانات نقطة المبيع | استيراد من POS خارجي | نقاط المبيع | ⚪ | يعتمد بناء POS | — |
| تصدير معلومات السيرفر | نسخ إعدادات/بيانات خادم | أدوات أخرى | ⚪ | سياسة أمان + نسخ احتياطي | — |
| استيراد معلومات السيرفر | استعادة إعدادات | أدوات أخرى | ⚪ | — | — |
| تصدير البيانات | تصدير عام (Excel/CSV) | أدوات، تقارير | ⚪ | واجهة تصدير لكل كيان رئيسي | — |
| استيراد البيانات | استيراد عام دفعات | أدوات | 🟡 | قوالب + تحقق أخطاء | حسابات، CC (دفعة) |
| استيراد البطاقة | استيراد بطاقات (حساب، عميل، مادة…) | مرجعيات | 🟡 | قالب لكل نوع بطاقة | حسابات، عملاء (يدوي) |

## تدفق البيانات

```
ملف (شركة/فرع)
    → مرجعيات (بطاقات)
    → معاملات (سندات، فواتير)
    → تصدير/أرشفة
```

## ملاحظات للتطوير

- **الأولوية المقترحة:** الفروع قبل POS والمواد متعددة الفروع.
- **استيراد البطاقة:** توحيد مع [أدوات](أدوات/README.md) لتجنب تكرار الواجهات.
