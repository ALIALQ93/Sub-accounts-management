# ملف

## الهدف الواجب الوصول إليه

إدارة **هوية المؤسسة** وبنيتها التنظيمية، وتمكين **نقل البيانات** بين الأنظمة والفروع — أساس كل الوحدات الأخرى.

## النتيجة المرجوة

- شركة (أو شركات) معرّفة ببيانات قانونية وشعار
- فروع مرتبطة بالحركات والتقارير
- استيراد/تصدير موحّد للبطاقات والبيانات التشغيلية

## العلاقات

### يعتمد على

- المصادقة والصلاحيات

### يغذّي

- [الحسابات](الحسابات/README.md) — فترات، فروع
- [المواد والمستودعات](المواد%20والمستودعات/README.md) — مواد الفروع
- [نقاط المبيع](نقاط%20المبيع/README.md) — مزامنة POS
- [أدوات](أدوات/README.md)

### يرتبط بـ

- `company_settings` في قاعدة البيانات
- `/settings/company`
