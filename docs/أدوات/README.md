# أدوات

## الهدف الواجب الوصول إليه

أدوات **صيانة البيانات**: استيراد، تصدير، إعادة ترحيل، ضغط — دون خلطها مع شاشات التشغيل اليومي.

## النتيجة المرجوة

- استيراد/تصدير موحّد لكل البطاقات
- صيانة آمنة بصلاحيات admin
- ربط مع [ملف](ملف/README.md)
