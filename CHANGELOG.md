# سجل التغييرات

جميع الإصدارات الملحوظة لهذا المشروع.  
**لتحديث الإصدار في الواجهة:** عدّل `web/src/config/app-release.ts` و`web/package.json` معاً.

## [0.1.0+trial-1] — 2026-07-04 — تجربة أولى

### قاعدة البيانات
- مخطط كامل عبر `database/setup_all.sql` (يشمل `06_storage.sql`)
- عملة وسعر صرف وقيمة أساسية على أسطر القيود (`patch_journal_line_currency.sql` مدمج)
- ترحيل تلقائي للسندات، صلاحيات، RLS، مرفقات السندات

### السندات
- قبض / صرف / تصفية مع عملة السند وسعر صرف محفوظ
- اعتماد وترحيل من قائمة السندات
- مزامنة قائمة الحسابات أثناء الإدخال
- قائمة بحث الحساب تظهر فوق جدول الأسطر

### التقارير
- كشف حساب: حسابات متعددة، عملة عرض، أعمدة القيمة الأصلية منفصلة
- استخدام القيم المخزنة على القيد (لا يعتمد على سجل أسعار بتاريخ الحركة)

### النظام
- إشعار التجربة الأولى عند الدخول (قابل للإغلاق)
- صفحة «عن البرنامج» مع ملاحظات الإصدار
- توثيق `database/TRIAL_SETUP.md` و`RELEASE_CHECKLIST.md`

### معروف / للمرحلة التالية
- ميزان المراجعة يجمع `debit`/`credit` وليس `debit_base`/`credit_base`
- القيود اليدوية لا تملأ حقول العملة الأساسية تلقائياً
